Something Weird!! [CSS Animation, no JS]

by Dirus

In this pen the author has created a complex animated image that responds to mouse hover using only CSS. The HTML consists of only a series of divs. These divs are each assigned one or more classes, and are selected with CSS in order to style each one individually. Different CSS parameters becoming active on hover. In order to animate the elements, they use the @keyframes rule along with the following CSS parameters:

  • transform
  • animation
  • border-color
  • border-radius
